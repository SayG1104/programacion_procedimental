Menú

El módulo de menú permite visualizar de forma clara los platos, bebidas y promociones disponibles.

Quienes Somos

El módulo ¿Quiénes somos? presenta la historia, valores y compromiso de la repostería SJ. Permite al usuario conocer nuestra identidad, calidad en los productos y dedicación al servicio.

Pedidos

El módulo de pedidos permite realizar solicitudes de productos de forma rápida y sencilla. El usuario puede seleccionar artículos, indicar cantidades y confirmar su compra cómodamente.

Usuarios

El módulo de usuarios permite registrarse, iniciar sesión y gestionar la información personal de forma segura. Facilita el acceso a pedidos, historial de compras y preferencias del cliente.

Inventario

El módulo de inventario permite controlar y consultar la disponibilidad de productos e insumos. Facilita la actualización de existencias para mantener una gestión ordenada y eficiente.

Contabilidad

El módulo de contabilidad permite registrar y controlar ingresos, gastos y movimientos financieros del negocio. Facilita una administración organizada para apoyar la toma de decisiones.
